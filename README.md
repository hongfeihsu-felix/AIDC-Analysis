# AIDC Analysis — AI Data Center Market Intelligence

## Report Scope
Bi-weekly market intelligence.

## Six Framework Dimensions
1. **AI Capex Cycle** — Capex trends across major CSPs (Google/Meta/Microsoft/Amazon/OpenAI/Anthropic)
2. **GPU/ASIC Supply & Demand** — H100/B100/GB200/Gaudi3/TPU supply-demand dynamics
3. **CSP Deep Dives** — AI infrastructure deployment cadence and capacity bookings per customer
4. **Interconnect & Cooling Tech** — NVLink/CPO/liquid cooling technology roadmaps
5. **HBM/Memory Cycle** — SK Hynix/Samsung/Micron supply-demand inflection points

## Data Sources

- Private (`data_private/`): TSMC earnings transcripts, expert interviews
